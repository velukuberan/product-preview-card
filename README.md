# Frontend Mentor - Product Preview Card Component

A responsive product preview card built with semantic HTML and CSS custom properties.

![Design preview for the Product preview card component coding challenge](./design/desktop-preview.jpg)

## Overview

This component displays a perfume product card with image, description, pricing, and an "Add to Cart" button. The layout adapts from a vertical stack on mobile to a horizontal layout on desktop.

## Built With

- Semantic HTML5
- CSS custom properties (design tokens)
- Flexbox
- BEM naming convention
- Mobile-first responsive design

## Features

- **Responsive layout**: Stacked on mobile, side-by-side on desktop (breakpoint: 768px)
- **Design tokens**: Centralized colors, spacing, and typography in CSS variables
- **Typography system**: Preset classes for consistent text styling
- **Accessible**: Proper alt text, semantic markup, and `aria-hidden` on decorative icons

## What I Learned

- Using CSS custom properties to create a maintainable design system
- Implementing BEM methodology for scalable CSS architecture
- Handling responsive images with different border-radius values per breakpoint
